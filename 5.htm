CSS background

CSS allows the use of solid colors as a background, and also allows the use of background images to create fairly complex effects.

 

CSS in this area is far above the ability of HTML.

background-color can not inherit its default value is transparent. transparent has the meaning of "transparent". That is, if an element does not specify a background color, the background is transparent so that the background of its ancestor element is visible.

Background repeat

If you need to tweak the background image on the page, you can use the background-repeat property.

The repeat value of the property causes the image to be tiled horizontally and vertically, just as usual for background images in the past. The repeat-x and repeat-y causes the image to repeat only horizontally or vertically, respectively, and no-repeat does not allow the image to be tiled in any direction.
