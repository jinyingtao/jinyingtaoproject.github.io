HTML introduction

What is HTML


HTML is a language used to describe web pages

-HTML refers to the Hyper Text Markup Language (Hyper Text Markup Language)

-HTML is not a programming language, but a markup language

-Markup language is a markup tag (markup tag)

-HTML uses tag tags to describe web pages

 

HTML tags


HTML tag tags are often referred to as HTML tags.

-HTML tags are keywords surrounded by angle brackets, such as <.html.>

-HTML tags usually appear in pairs, such as <.b> and <./b>

-The first label in the label pair is the start label and the second label is the end label

-The start and end tags are also known as open tags and closed tags

 

HTML document = webpage


-HTML document description page

-HTML documents contain HTML tags and plain text

-HTML documents are also called web pages
