HTML & CSS

Hypertext Markup Language, Chinese is the hypertext markup language. HTML (HyperTextMark-upLanguage) that Hypertext Markup Language, WWW description language. The purpose of designing the HTML language is to make it easy to associate texts or graphics stored in one computer with texts or graphics in another computer to form an organic whole without regard to the fact that the specific information is on the current computer Still on other computers on the network. We just use the mouse to click an icon in a document, the Internet will immediately go to the contents of this icon up, and the information may be stored in another computer network.

 
HTML text is a descriptive text made up of HTML commands that describe text, graphics, animations, sounds, tables, links, and more. The structure of HTML includes the head and the body. The head describes the information required by the browser, and the body contains the specific content to be described.
 
The CSS language is the acronym for "Cascading Style Sheets," a Chinese translation of "Cascading Style Sheets," a web publishing standard developed and promulgated by the W3C Association to complement HTML language features. The main purpose is to control the fonts, colors, backgrounds, images, and other elements of the web page so that the web page can be displayed exactly as the designer asks. The CSS language is a markup language for web page layout.
