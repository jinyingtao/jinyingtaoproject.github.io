CSS syntax


The CSS rule consists of two main parts: the selector, and one or more statements.

 

The selector is usually the HTML element that you need to change the style.

Each statement consists of a property and a value.

Property is the style attribute that you want to set. Each property has a value.

Attributes and values are separated by a colon.
