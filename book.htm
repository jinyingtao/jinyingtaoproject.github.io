Good book

Here are some good books, I will recommend to you

1. H1, HTML5 articles - "HTML5 and CSS3 basic tutorial (eighth edition)"

The HTML5 and CSS3 Basic Tutorial (8th Edition) is a classic bestseller for introductory knowledge of HTML and CSS. It provides a thorough and systematic overview of the basics of HTML5 and CSS3, as well as practical techniques, and analyzes many aspects of web page production in large numbers. The book not only introduces the webpage elements such as texts, images, links, lists, forms, forms, etc .; also introduced how to design the layout for the webpage, add the dynamic effect etc .; In addition, also involve to debug and release. This book has provided a formidable supporting website, has listed above the complete code example in the book, of course, more outstanding examples and the advanced reference materials are available, for the reader reference study.

 

2. HTML5 - "HTML5 and CSS3 examples tutorial (2nd Edition)"

The HTML5 & CSS3 Sample Tutorial (Version 2) discusses the real issues to be solved for the latest version of the browser. Each chapter focuses on a specific problem, introduces specific concepts through multiple instances, and guides the reader through the creation of simple examples to arrive at a solution. Instance followed by the "back-off program" for the old version of the browser to provide solutions. The Future Outlook explores how to better use the new features that are now mature in the future. The complete code for all instances is provided free of charge.

 

3. HTML5 - HTML5 authoritative guide

By far the most comprehensive and detailed web design reference book Wonderful rendering of more than 500 actual code examples and mainstream browser renderings
Intimate compilation of concise reference tables for all the attributes, elements, and functions in HTML5 and CSS3
? The HTML5 authoritative guide is the authoritative reference book for system learning web design. The book is divided into five parts: The first part introduces the preparatory knowledge of this book and the latest development of HTML, CSS and JavaScript; the second part discusses the HTML elements and details the new and modified elements in HTML5; The third part explains CSS, which covers all the CSS selectors and attributes that control the style of content, supplemented by a large number of code examples and illustrations; the fourth part introduces the DOM, an analysis of how to manipulate HTML content with JavaScript; the fifth part explains Ajax, multimedia, and canvas elements HTML5 advanced features.
